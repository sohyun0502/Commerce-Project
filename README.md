# 🛒 상품 관리 시스템 (Commerce Management System)

Java 콘솔 환경에서 동작하는 **상품 관리 및 구매 시스템**입니다.  
관리자와 고객 역할을 분리하여,
- 관리자는 상품을 관리하고
- 고객은 등급을 선택하여 할인 혜택을 받을 수 있도록 설계되었습니다.

객체지향 설계를 기반으로 **상품, 카테고리, 장바구니, 고객, 관리자**의 책임을 분리하여 구현했습니다.

```
commerce/
 ├── Administrator.java     # 관리자 기능 (상품 CRUD)
 ├── Customer.java          # 고객 정보 및 등급 선택

```

## 👤 사용자 역할

### 🔐 관리자 (Administrator)

- 비밀번호 인증 기반 접근
- 상품 관리 전용 기능 제공

#### 관리자 인증
- 비밀번호: `admin123`
- **3회 실패 시 메인 메뉴로 이동**

---

### 🧑 고객 (Customer)

- 고객 등급 선택
- 등급에 따라 할인율 적용

#### 고객 등급 선택
```
1. BRONZE : 0% 할인
2. SILVER : 5% 할인
3. GOLD   : 10% 할인
4. VIP    : 15% 할인
```

> 고객 등급은 `CustomerLevel` Enum으로 관리되며  
> 각 등급은 고정된 할인율을 가집니다.

---

## 📦 상품 관리 기능 (관리자 전용)

### 1️⃣ 상품 추가
- 카테고리 선택
- 상품 정보 입력
    - 상품명
    - 가격
    - 설명
    - 재고 수량
- 입력 정보 확인 후 등록

### 2️⃣ 상품 수정
- 상품명으로 검색
- 수정 가능 항목
    - 가격
    - 설명
    - 재고 수량

### 3️⃣ 상품 삭제
- 상품명 기준 삭제
- 삭제 시 **장바구니에 담긴 상품도 함께 제거**

### 4️⃣ 전체 상품 조회
- 카테고리별 상품 목록 출력
- 상품이 없는 경우 안내 메시지 출력

```
[ 전자제품 ]

[ 식품 ]
등록된 상품이 없습니다.
```

## 💡 주요 구현 포인트

### ✔ 역할 분리
- `Administrator`: 관리자 인증 및 상품 관리
- `Customer`: 고객 정보 및 등급 선택
- `CustomerLevel`: Enum 기반 할인 정책
- `Category`: 카테고리별 상품 관리
- `Product`: 상품 데이터 관리
- `Cart`: 장바구니 관리

### ✔ Enum 활용
- 고객 등급을 Enum으로 관리하여
    - 할인율 변경 시 유지보수 용이
    - 가독성과 안정성 향상

### ✔ 데이터 일관성
- 상품 삭제 시 장바구니에서도 자동 제거
