# 🧮 커머스 과제

이 프로젝트는 **Java 문법 과제**로,
- `switch` `while`
- `클래스`
- `예외처리`
- `컬렉션`
- `Enum`
- `제네릭(Generic)`
- `Lambda & Stream`

을 실제 코드에 적용해보는 것을 목표로 한다.

---

## 📌 과제 요구사항
### step1
#### 1️⃣ 양의 정수(0 포함) 입력받기
- Scanner를 사용하여 입력받기
- 양의 정수이므로 `int`타입으로 변수 선언

#### 2️⃣ 사칙연산 기호(➕,➖,✖️,➗) 입력받기
- Scanner를 사용하여 입력받기
- `charAt(0)`을 이용하여 타입 설정

#### 3️⃣ 입력받은 양의 정수 2개와 사칙연산 기호를 사용하여 연산 진행
- `switch`문을 이용해 연산기호 구분
- "0으로 나누기" 라는 연산 오류 예외처리

#### 4️⃣ 반복문을 사용하되, “exit” 문자열을 입력하기 전까지 무한으로 계산 진행
- `while(true)`를 사용하여 무한루프 실행

### step2
#### 1️⃣ 사칙연산 수행 후 결과값 반환 메서드 구현 & 연산 결과 저장하는 컬렉션 타입 필드를 가진 Calculator 클래스 생성
- public 반환타입 calculate(...매개변수) { } 형태로 구현
- ArrayList<Integer> 컬렉션 만들기

#### 2️⃣ main 메서드에 Calculator 클래스 활용

#### 3️⃣ main 메서드에서 Calculator 클래스의 컬렉션 필드에 직접 접근 못하도록 수정 (캡슐화)
- Getter, Setter 메서드 구현
- private 접근제어자 사용하여 캡슐화

#### 4️⃣ 컬렉션 필드에 저장된 숫자들 중 가장 먼저 저장된 데이터를 삭제하는 메서드 구현
- public void removeResult() { } 형태로 구현
- remove() 사용하여 컬렉션의 데이터 삭제

### step3
#### 1️⃣ Enum을 활용한 연산자 관리
- 사칙연산(➕, ➖, ✖️, ➗)을 `enum` 타입으로 정의
- 연산자에 대한 정보를 한 곳에서 관리
- 잘못된 연산자 입력 시 예외 처리

#### 2️⃣ 제네릭을 사용한 계산기 확장
- `double`을 포함한 **여러 숫자 타입**을 처리할 수 있도록 개선
- 단순 타입 변경이 아닌 **제네릭(Generic)** 사용

#### 3️⃣ Lambda & Stream을 활용한 결과 조회
- 사용자로부터 입력받은 값보다 **큰 결과들만 필터링하여 출력**
- **Stream + Lambda 표현식** 사용

## 📂 프로젝트 구조

```text
step1
 ┗ App.java                     // 프로그램 실행 (입력/출력 담당)
step2
 ┣ App.java                     // 프로그램 실행 (입력/출력 담당)
 ┗ Calculator.java              // 계산기 클래스
step3
 ┣ App.java                     // 프로그램 실행 (입력/출력 담당)
 ┣ ArithmeticCalculator.java     // 제네릭 계산기 클래스
 ┗ OperatorType.java             // 연산자 Enum
```

## ▶️ 실행 방법

1. 프로그램 실행
2. 첫 번째 숫자 입력
3. 두 번째 숫자 입력
4. 연산자 입력 (`+`, `-`, `*`, `/`)
5. 결과 출력
6. `exit` 입력 시 종료
